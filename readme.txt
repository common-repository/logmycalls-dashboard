View LogMyCalls call metrics on the WP Dashboard.

== Description ==

Easily track LogMyCalls call metrics from the comfort of your WP Dashboard. After entering your LogMyCalls API Credentials (Requires LogMyCalls *Enterprise* service) choose from a list of metrics to show over the past day/lifetime:

* Total Calls
* Total Calls Answered
* Total Calls Unanswered
* Longest Call Duration
* Total Call Duration
* Average Call Duration

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit Settings -> LogMyCalls Settings to configure the plugin.

== Frequently Asked Questions ==

= Can I use this plugin with LogMyCalls Free, Basic, or Pro? =

Not at this point. LogMyCalls API connectivity is only available with LogMyCalls *Enterprise*
